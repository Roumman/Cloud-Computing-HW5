# ДЗ 5 (load testing & monitoring)

### Конфигурация нагрузочного тестирования для Yandex Tank

Используется генератор Phantom, конфигурация и патроны лежат в `/monitoring`.

### Экспорт метрик сервисом

Сервис экспортирует следующие prometheus метрики по стандартному эндпоинту `GET /metrics`:

* `requests_total` - общее количесвто запросов по URL, методу и статусу ответа;
* `style_transfer_requests_total` - количество запросов на преобразование изображения по каждой модели;
* `request_latency_seconds` - время выполнения запроса по URL и методу.


### Мониторинг

Конфигурация Prometheus на машине для мониторинга - `monitoring/prometheus.yml`.

Для отслеживания состояния метрик из Node Exporter используется стандартный дашборд.

Для визуализации метрик по самому сервису (RPS, latency) создан дашборд `monitoring/grafana_dashboard.json`.
